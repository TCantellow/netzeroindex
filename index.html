The Net Zero Vulnerability Index (NZVI) is a tool to assess how vulnerable different regions in the UK are to economic changes as the country moves towards a net-zero economy. This model considers various factors, such as local economic complexity and relatedness, how many jobs rely on at-risk sectors, and the size of the local population, to create a profile that shows which areas might be most affected by changes related to achieving net-zero emissions.

A higher score indicates an area more at risk. This interactive map presents data for the following administrative geographies:

  • LAD: Local Authority Districts.
  • LEP: Local Enterprise Partnerships.
  • CA: Combined Authorities1.

The NZVI goes beyond just measuring job losses; it also considers how well communities and economies can adapt to changes in traditional industries. By examining the complexity of jobs, the connections between different industries, reliance on key sectors, and how many people might be affected, the NZVI provides a detailed picture of regional vulnerabilities.

1 This index uses boundary data from 2022, before the boundaries were updated in 2024, as underlying data used to construct the index is not yet available..
